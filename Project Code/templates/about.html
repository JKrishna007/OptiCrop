How OptiCrop Works

A deep dive into the methodology behind our crop recommendation engine

📊
Data-Driven

Trained on 2,200+ agricultural samples spanning 22 different crops with 7 environmental features.

🤖
ML-Powered

Compares 5 algorithms — KNN, Logistic Regression, Decision Tree, Random Forest, and Naive Bayes — and selects the best.

⚡

Sub-second predictions served through a lightweight Flask backend with pre-loaded model artifacts.

Methodology

Input Parameters
  • Nitrogen (N) — Ratio of nitrogen content in soil (kg/ha)
  • Phosphorous (P) — Ratio of phosphorous content in soil (kg/ha)
  • Potassium (K) — Ratio of potassium content in soil (kg/ha)
  • Temperature — Average temperature (°C)
  • Humidity — Relative humidity (%)
  • pH Level — Soil acidity/alkalinity (0–14)
  • Rainfall — Annual rainfall (mm)
ML Pipeline
  1. Load & clean the agricultural dataset
  2. Handle missing values & outliers
  3. Encode crop labels with LabelEncoder
  4. Scale features with StandardScaler
  5. Split into 80% train / 20% test (stratified)
  6. Train 5 algorithms & evaluate metrics
  7. Select the best-performing model
  8. Persist model + scaler + encoder as .pkl

Algorithms Compared

Algorithm Type Key Strength
K-Nearest Neighbors Instance-based Simple, effective for small datasets
Logistic Regression Linear classifier Interpretable probability estimates
Decision Tree Tree-based Highly interpretable, captures non-linear patterns
Random Forest ⭐ Ensemble Best accuracy; robust to overfitting
Naive Bayes Probabilistic Very fast; good baseline classifier
